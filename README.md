# 🧪 Laboratorio de Programación Básica (JS)

Este repositorio contiene el desarrollo de un laboratorio enfocado en los fundamentos de programación, aplicando lógica mediante diferentes ejercicios prácticos.

## 📚 Temas abordados

* Variables y tipos de datos
* Funciones
* Condicionales (`if / else`)
* Bucles (`for`, `while`)
* Arrays y métodos básicos

---

## 🎯 Objetivo

Aplicar los conceptos básicos de programación mediante la resolución de problemas comunes, simulando situaciones reales como sistemas de inventario, aplicaciones bancarias y plataformas educativas.

---

## 🧩 Ejercicios

### 1. Calculadora de Propinas 🍽️

Función: `calcularPropina(total, porcentaje)`
Calcula el monto total a pagar incluyendo la propina.

---

### 2. Validar Contraseña 🔐

Función: `validarContraseña(contraseña)`
Valida que la contraseña:

* Tenga al menos 8 caracteres
* Incluya un número
* Incluya una letra mayúscula

---

### 3. Buscar Producto en Inventario 📦

Función: `buscarProducto(nombre, inventario)`
Busca un producto dentro de un inventario predefinido y retorna el objeto o `null`.

Ejemplo de inventario:

```js
const inventario = [
  { nombre: "Camisa", precio: 20 },
  { nombre: "Zapatos", precio: 50 }
];
```

---

### 4. Calcular Promedio de Notas 📊

Función: `calcularPromedio(notas)`
Recibe un array de notas ingresadas por el usuario y retorna el promedio (redondeado a 2 decimales).

---

### 5. Filtrar Usuarios por Edad 👥

Función: `filtrarUsuarios(usuarios, edadMinima)`
Retorna los usuarios que cumplen con la edad mínima indicada.

---

### 6. Contador de Palabras ✍️

Función: `contarPalabras(texto)`
Cuenta la cantidad de palabras en un texto.

---

### 7. Simulador de Cajero Automático 💳

Función: `retirarDinero(saldo, monto)`
Permite retirar dinero y valida si hay fondos suficientes.

---

### 8. Cambio de Moneda 💱

Función: `convertirMoneda(monto, monedaDestino)`
Convierte pesos colombianos (COP) a USD o EUR según la opción del usuario.
